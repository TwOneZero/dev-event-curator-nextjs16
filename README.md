# DevEvent (dev-event-nextjs)

Next.js 16을 활용한 FullStack 개발 학습 및 개발 이벤트 큐레이팅 서비스입니다.

## 🚀 프로젝트 소개
이 프로젝트는 각 개발 관련 기업이나 플랫폼에서 진행(또는 예정) 중인 이벤트를 한눈에 확인할 수 있도록 큐레이팅해주는 사이트입니다. 
단순한 정보 제공을 넘어 실제 프로덕트 수준의 환경을 구성하기 위해 **PostHog**를 도입하여 사용자 행동 모니터링 및 분석을 수행합니다.

## 🎯 주요 목적
- **Next.js 16 (App Router) & React 19 학습**: 최신 기술 스택을 활용한 FullStack 웹 애플리케이션 구축 연습.
- **실무 환경 경험**: PostHog를 통한 데이터 기반의 모니터링 및 분석 환경 구축.
- **이벤트 큐레이션**: 흩어져 있는 개발 이벤트 정보를 한곳에 모아 사용자에게 제공.

## 🛠 기술 스택
- **Framework**: Next.js 16.1.4 (App Router)
- **Database**: MongoDB,
- **Library**: React 19, Shadcn(lucide react) for react-bits component
- **Styling**: Tailwind CSS 4, class-variance-authority
- **Analytics & Monitoring**: PostHog
- **Language**: TypeScript

## ✨ 주요 기능
- **개발 이벤트 큐레이션**: 해커톤, 밋업, 컨퍼런스 등 다양한 개발 행사 정보 제공.
- **사용자 분석 (PostHog)**: 페이지 뷰 및 사용자 인터랙션 트래킹을 통한 프로덕트 분석.
- **반응형 디자인**: 다양한 디바이스에 최적화된 UI/UX 제공.

## ⚙️ 시작하기

### 설치
```bash
npm install
```

### 실행
```bash
npm run dev
```
